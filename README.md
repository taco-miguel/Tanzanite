
# Tanzanite Mod

The mod was supposed to be for a project I have been working on - SatisSMP, a medieval roleplay server. The project kinda didnt work out but I dont want to delete this since i think the shader included in it is kinda cool. 75% of the mod is written by myself, the shader is vibecoded tho because at that time I didnt have capabilities to code one myself. I am learning tho!!


## Features

- Basic blocks like a normal ore, deesplate variant etc
- Method to obtain Tanzanite Powder which can be put into a pipe and smoked
- Cool shader on indulge
- Cool crystal crushing process
